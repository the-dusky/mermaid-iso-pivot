# Dev Notes

Research, experiments, and implementation notes that support development but don't fit the ADR format.

## Purpose

Use dev notes for:
- Research on libraries and tools
- Experiment results
- Performance benchmarks
- Implementation spikes
- Debugging notes

## vs ADRs

| ADR | Dev Note |
|-----|----------|
| Formal decision record | Informal notes |
| Follows template | Free-form |
| Tracks alternatives | Tracks research |
| Has status lifecycle | Snapshot in time |
